# Fake-News-Detection-With-Bert
## Project Description
Detect fake news from title by training a model using Bert.
## How to run the project?

```sh
jupyter notebook
```
and then run Fake-News-Detection-With-Bert.ipynb
or 
```sh
python Fake-News-Detection-With-Bert.py
```
